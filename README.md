# 📊 Data Analyst Agent

An AI-powered **Data Analyst Agent** built using **Google Agent Development Kit (ADK)** and **Gemini** that can write, execute, and verify Python code to analyze data, solve mathematical problems, and generate data-driven insights automatically.

---

## 🚀 Features

✨ Analyze datasets using Python  
📈 Perform statistical calculations and data analysis  
🧮 Solve mathematical and logical problems with code verification  
📊 Generate insights from raw data  
📉 Describe charts and visualizations  
⚡ Execute Python code automatically before responding  
🤖 Powered by Google Gemini + ADK  

---

## 🛠️ Tech Stack

- **Python**
- **Google ADK (Agent Development Kit)**
- **Gemini 3 Flash Preview** #free API key not paid on Google studio.
- **Built-in Code Execution Environment**
